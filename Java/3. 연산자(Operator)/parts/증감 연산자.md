##### > 증감 연산자
>[[증감 연산자|증감]] 연산자
> 증가 연산자(++) : 피연산자의 값을 1 증가시킨다.
> 감소 연산자(--) : 피연산자의 값을 1 감소시킨다.

|  타입  | 설명                               | 사용 예시 |
|:------:| ---------------------------------- |:---------:|
| 전위형 | 값이 참조되기 ==전에== 증감시킨다. | j = ++i;  |
| 후위형 | 값이 참조된 ==후에== 증감시킨다.   | j = i++;  |
>표 1: 전위형과 후위형의 비교

###### 1. 전위형
```java
int i = 5;
int j = ++i;
```
1. i의 값을 1증가(5 -> 6)
2. i (6)의 값을 참조
3. 연산결과를 j (6)에 저장함

###### 2. 후위형
```java
int i = 5;
int j = i++;
```
1. i의 값을 참조
2. 연산결과를 j (6)에 저장함
3. i의 값을 1증가(5 -> 6)
